First program - Hello world!

The first program to write is the same for all languages, print the words Hello world!.

In the C programming language, the program to print Hello world! is

                    
#include <stdio.h>
                
int main()
{
    printf("Hello world!\n");
    return 0;
}

Compile

cc hello.c -o hello

Run

./hello

Hello world!
